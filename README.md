# my-pet-project

Такое вот веб‑приложение для хранения и поиска химических молекул.

Субструктурный поиск химических соединений — важнейший инструмент в хемоинформатике. 
С его помощью исследователи и интересующиеся могут идентифицировать и анализировать химические молекулы по определенным субструктурам. 
Этот метод применяют в различных областях химии, среди которых разработка лекарств, материаловедение и экологи ческие исследования.
Современные базы данных химических соединений содержат миллионы записей, что делает традиционные методы поиска неэффективными и трудоемкими. 
Субструктурный поиск использует алгоритмы, позволяющие быстро и точно идентифицировать соединения с заданными структурными фрагментами.

# SMILES

Ключевым элементом в представлении химических структур является упрощенная система ввода молекулярных линейных записей (SMILES — Simplified Molecular Input Line Entry System). 
SMILES — это нотация, позволяющая представлять химическую структуру в виде, который легко обрабатывается компьютерами.

## Что тут дают?

- Добавление, просмотр, обновление и удаление молекул
- Субструктурный поиск с использованием RDKit
- Кеширование результатов поиска в Redis
- Асинхронный поиск через Celery
- Веб‑интерфейс на HTML/CSS/JS с пагинацией и двумя режимами поиска
- Полная контейнеризация всех сервисов с Docker
- CI/CD пайплайн на GitHub Actions
